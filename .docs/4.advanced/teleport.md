Vue 3 提供了 [`<Teleport>` 组件](https://vuejs.org/guide/built-ins/teleport.html)，它允许内容在 DOM 中的其他地方渲染，超出 Vue 应用程序的范围。

此示例展示了如何在客户端渲染和服务器端渲染中使用 `<Teleport>`。
